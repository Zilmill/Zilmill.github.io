Electron 集成 Sqlite3 问题

前言

在 Electron 中集成 Sqlite3 时遇到的问题。

首先要确认Python的版本对不对，python 指向的必须是 python2.x版本

安装node-gyp

安装 Sqlite3

添加electron-builder

修改package.json ，这个添加的话，在 npm i安装的时候就会自动触发

如果不是 npm i 需要手动触发

全部安装后，需要手动更改文件名称

删除 node-v64-darwin-x64 文件夹，重命名 electron-v6.0-darwin-x64 为 node-v64-darwin-x64

如果编译失败，请检查python 指向的是否是 Python2.x (2.5+)

然后就可以直接启动了

npm run start

Sqlite3 必须在主进程中使用，因为有操作本地二进制文件
